# Project Brief: 대동 (Daedong) AI-Powered Business Portal

**Version:** 1.0
**Date:** 2025-11-07
**Status:** Draft

---

## Executive Summary

대동티엘 & 대동라이즈는 AI를 전면 도입하여 **'24시간 스스로 영업하고 마케팅하는 능동형 비즈니스 엔진'**을 구축합니다. 본 프로젝트는 듀얼 트랙으로 구성되며, Track A는 고가치 B2B 파트너십 유치를 위한 AI 웹 포털, Track B는 국내외 이커머스 채널의 지능형 운영 자동화를 목표로 합니다.

**핵심 가치 제안:** AI가 웹사이트 방문자를 실시간 분석하여 잠재 파트너를 식별하고, 이커머스 채널을 24시간 모니터링하여 CS, 위기 대응, 상세 페이지 최적화를 자동 수행함으로써 최소 인력으로 글로벌 시장 확장을 가속화합니다.

**주요 시장:** B2B 글로벌 유통 파트너 (3M 산업용 솔루션, 전동공구), B2C/소규모 B2B 온라인 구매자 (쿠팡, 스마트스토어, Amazon, Shopify)

## Problem Statement

### Current State & Pain Points

**기존 정적 웹사이트의 한계:**
- 방문자 분석 없이 일방향 정보만 제공
- B2B 리드 발굴이 수동적이며 기회 손실 발생
- 글로벌 파트너의 문의에 즉각적인 대응 불가
- 견적 요청 처리가 느리고 비효율적

**수동적 이커머스 운영의 문제:**
- CS 문의 대응에 인력 과다 투입 (단순 반복 업무 90%)
- 부정 리뷰 및 위기 고객 감지 지연으로 평판 손상
- 경쟁사 가격/재고 변동 대응 시차로 매출 기회 상실
- 상세 페이지가 고객 피드백을 반영하지 못해 전환율 저하

### Impact & Quantification

- **영업 기회 손실:** 해외 바이어가 웹사이트 방문 후 즉각적인 대응 없이 이탈
- **CS 비효율:** 단순 문의(배송, 사이즈) 대응에 업무 시간의 80% 이상 소요
- **위기 대응 지연:** 부정 리뷰 감지까지 평균 24시간 소요, 공개 비난 확산 위험
- **경쟁 열위:** 경쟁사 품절 시 기회 포착 실패로 시장 점유율 상실

### Why Existing Solutions Fall Short

- **기존 챗봇:** 단순 FAQ 응답에 그치며 리드 자격 부여(Qualification) 불가
- **수동 모니터링:** 다수 이커머스 채널을 사람이 감시하는 것은 비현실적
- **정적 콘텐츠:** 방문자 맥락(IP, 언어, 행동)에 따른 개인화 부재
- **번역 도구:** 현지화된 마케팅이 아닌 단순 번역으로 글로벌 경쟁력 부족

### Urgency

글로벌 B2B 시장과 온라인 이커머스 시장은 이미 AI 기반 자동화로 전환 중입니다. 지금 행동하지 않으면 경쟁사 대비 영업·마케팅 효율성에서 뒤처지며, 특히 3M 공식 파트너십이라는 차별화 요소를 충분히 활용하지 못하게 됩니다.

## Proposed Solution

### Core Concept

**AI 기반 듀얼 트랙 비즈니스 엔진**을 구축하여, 웹사이트는 능동적으로 B2B 리드를 발굴·전환하고, 이커머스 채널은 AI가 24시간 모니터링하며 자동 대응하는 통합 시스템을 만듭니다.

### Track A: AI B2B 웹 포털

**1. 24/7 AI 세일즈 에이전트 (AI Sales Agent Chatbot)**
- 방문자의 페이지 체류, IP, 언어를 실시간 분석
- '3M 산업용' 페이지 30초 이상 체류 시 맥락 기반 대화 시작
- 적용 분야, 회사명, 연락처를 수집하여 고가치 리드 확보

**2. 실시간 웹 개인화 엔진 (Real-time Personalization Engine)**
- 베트남 IP 접속 시 메인 배너를 '베트남 파트너십 성공 사례'로 자동 변경
- '전동공구' 페이지 반복 방문자에게 관련 특가 섹션 노출
- 이탈 방지 및 전환율 극대화

**3. 지능형 견적 시스템 (Intelligent Quote Generator)**
- 대화형 폼으로 제품, 수량, 국가 정보 수집
- 학습된 데이터 기반 예상 견적 범위 즉시 제공
- "정확한 공식 견적서를 받으시려면 이메일을 남겨주세요" → B2B 리드 확보

**4. AI 기반 연관 추천 (Proactive Recommendation Engine)**
- 'DeWALT 드릴' 페이지에서 "3M 보안경과 작업용 장갑도 함께 구매" 추천
- 교차 판매(Cross-selling) 유도

### Track B: AI 이커머스 운영 엔진

**1. E-Commerce 모니터링 시스템 (Foundation)**
- 10분 간격으로 자사 스토어(쿠팡, 스마트스토어) 및 경쟁사 스토어 스크래핑
- 신규 리뷰, 고객 문의(Q&A), 가격 변동, 재고 상태 자동 감지
- AI가 텍스트 분석(감성, 키워드)하여 변경 사항 파악

**2. AI 기반 지능형 고객 응대 (CS Automation)**
- 신규 문의/리뷰를 '긍정 / 단순 문의 / 부정 및 위기'로 자동 분류
- 단순 문의(배송, 사이즈): AI가 90% 완성도 답변 초안 생성 → 담당자 전송만 클릭
- 부정 및 위기: 즉시 슬랙/메신저 알림 발송

**3. 위기(Critical) 고객 적극 대응 SOP**
- 공개 댓글 대신 1:1 메시지로 개별 연락 (공개 비난 차단)
- 신속 교환/환불 + 소정 보상 제공으로 부정 경험을 긍정 경험으로 전환
- 불량 사유를 태그로 저장하여 상세 페이지 개선에 활용

**4. '살아있는' 상세 페이지 (AI-based CRO)**
- 매일 자정 모든 리뷰 데이터 수집 및 AI 분석
- 고객 핵심 키워드(예: "수면", "소음 차단") 빈도 추출
- F.A.Q 자동 생성 및 상세 페이지 업데이트
- A/B 테스트용 문구/이미지 제안

**5. 경쟁사 분석 및 기회 포착**
- 경쟁사 가격 5% 인하 시 즉시 알림
- 경쟁사 품절 감지 시 즉시 알림 (고급 기능: 자동 광고비 증액)

**6. 글로벌 이커머스 확장 (Amazon, Shopify)**
- AI 현지 키워드 리서치 및 현지화(Localization)
- 단순 번역이 아닌 현지 마케팅 최적화 카피라이팅
- 규제/인증 정보 자동 확인 및 강조

### Key Differentiators

- **능동형 AI 영업:** 방문자를 기다리는 것이 아니라, AI가 먼저 식별하고 유도
- **실시간 모니터링:** 10분 간격 감시로 경쟁사 대비 빠른 대응
- **데이터 기반 최적화:** 고객 리뷰를 실시간 반영하여 상세 페이지 지속 개선
- **3M 공식 파트너십:** 신뢰도 및 차별화 요소를 AI가 적극 활용

### Why This Will Succeed

기존 솔루션은 단순 자동화(FAQ 챗봇, 번역)에 그쳤지만, 본 프로젝트는 **AI가 비즈니스 의사결정의 주체**가 되어 리드 자격 부여, 위기 감지, 기회 포착까지 수행합니다. 또한 듀얼 트랙 구조로 B2B와 B2C 양쪽 시장을 동시에 공략하여 매출 다변화를 실현합니다.

## Target Users

### Primary User Segment: B2B 해외 파트너 (International Distributors & Buyers)

**Demographic/Firmographic Profile:**
- 지역: 동남아시아(베트남, 태국), 유럽, 북미
- 기업 규모: 중소형~대형 유통사, 산업재 바이어
- 의사결정자: 구매 담당자(Procurement Manager), 사업 개발 담당자(BD Manager)

**Current Behaviors & Workflows:**
- 신규 공급업체 발굴 시 웹사이트 방문 → 회사 신뢰도 확인 → 이메일 문의 → 견적 대기
- 3M 공식 유통 여부, 공급 능력(Capacity), 과거 실적 중시
- 대량 주문(Bulk Order) 견적 비교 후 파트너 선정

**Specific Needs & Pain Points:**
- 즉각적인 견적 범위 확인 필요 (대기 시간 단축)
- 공급업체의 신뢰도 및 인증 정보 명확한 확인
- 언어 장벽 (영어, 현지어 지원 필수)
- 시차로 인한 실시간 소통 어려움

**Goals:**
- 신뢰할 수 있는 3M 공식 파트너 발굴
- 경쟁력 있는 가격의 대량 견적 확보
- 빠르고 투명한 커뮤니케이션

### Secondary User Segment: B2C/소규모 B2B 온라인 구매자

**Demographic Profile:**
- 지역: 국내(한국) 우선, 향후 글로벌 확장
- 연령: 25-50세
- 직업: 건설업 종사자, 제조업 종사자, DIY 애호가, 학생(이어폼 등)

**Current Behaviors & Workflows:**
- 쿠팡, 스마트스토어에서 제품 검색 → 리뷰 확인 → 구매
- 정품 여부, 가격, 배송 속도 중시
- CS 문의 빈번 (사이즈, 배송, 사용법)

**Specific Needs & Pain Points:**
- 정품 확인 및 신뢰도 보장 필요
- 빠른 CS 응답 (24시간 이내)
- 상세한 제품 정보 및 사용 사례

**Goals:**
- 합리적인 가격의 정품 구매
- 신속한 배송 및 친절한 CS
- 제품 사용 후 만족도 확보

## Goals & Success Metrics

### Business Objectives

- **B2B 리드 자동화:** 웹사이트 방문자의 B2B 리드 전환율 **30% 증가** (런칭 후 6개월 내)
- **CS 자동화율:** 단순 문의 자동 응답률 **90% 달성** (런칭 후 3개월 내)
- **위기 대응 시간 단축:** 부정 리뷰 감지 및 1차 대응까지 **평균 1시간 이내** (현재 24시간 → 1시간)
- **글로벌 매출 증대:** Amazon, Shopify 채널 매출 **월 $10,000 달성** (런칭 후 12개월 내)
- **경쟁사 대비 대응 속도:** 가격/재고 변동 감지 후 대응까지 **30분 이내**

### User Success Metrics

- **B2B 파트너:** 견적 요청 제출률 증가, 견적 제공까지 소요 시간 감소
- **B2C 구매자:** CS 문의 응답 시간 감소, 구매 후 만족도(리뷰 평점) 4.5/5.0 이상 유지
- **웹사이트 방문자:** 평균 체류 시간 증가, 페이지 이탈률 감소

### Key Performance Indicators (KPIs)

- **B2B Lead Conversion Rate:** 웹사이트 방문자 중 연락처 정보 제공 비율 (목표: 5% → 7%)
- **CS Response Time:** 첫 응답까지 평균 시간 (목표: 12시간 → 1시간)
- **Negative Review Mitigation Rate:** 부정 리뷰의 긍정 전환 비율 (목표: 50%)
- **AI Chatbot Engagement Rate:** 챗봇 대화 시작 방문자 비율 (목표: 20%)
- **E-commerce Monitoring Coverage:** 모니터링 대상 스토어 수 및 감지 정확도 (목표: 95%)
- **Cross-selling Conversion:** AI 추천으로 인한 추가 구매 비율 (목표: 10%)

## MVP Scope

### Core Features (Must Have)

**Track A: AI B2B 웹 포털**

- **24/7 AI 세일즈 챗봇:**
  - 페이지 체류 시간 감지 (30초 이상)
  - 기본 리드 자격 부여 질문 (적용 분야, 회사명, 연락처)
  - 수집된 리드 데이터 저장 및 알림
  - **Rationale:** 핵심 B2B 리드 발굴 기능, 모든 자동화의 시작점

- **기본 웹 개인화:**
  - IP 기반 국가 감지 및 언어 전환 (한국어/영어)
  - 메인 배너 A/B 테스트 (글로벌 vs. 이커머스 퍼널)
  - **Rationale:** 글로벌 파트너 타겟팅에 필수

- **지능형 견적 시스템 (Beta):**
  - 대화형 폼으로 제품, 수량, 국가 정보 수집
  - 예상 견적 범위 제공 (사전 학습된 가격 데이터 기반)
  - **Rationale:** B2B 파트너의 즉각적인 니즈 충족

- **기본 사이트 구조 (IA):**
  - HOME, ABOUT US, GLOBAL BUSINESS, PRODUCTS, E-COMMERCE, CONTACT
  - 3M 공식 파트너십 인증 강조
  - **Rationale:** 신뢰도 구축 및 정보 제공

**Track B: AI 이커머스 엔진**

- **E-Commerce 모니터링 시스템 (Foundation):**
  - 쿠팡, 스마트스토어 2개 채널 모니터링
  - 신규 리뷰, Q&A 10분 간격 감지
  - 경쟁사 1개 스토어 가격/재고 모니터링
  - **Rationale:** 모든 자동화의 기반, 데이터 수집 시작

- **AI 기반 CS 자동화 (Phase 1):**
  - 문의 자동 분류 (긍정/단순/부정 및 위기)
  - 단순 문의 AI 답변 초안 생성
  - 위기 고객 즉시 알림 (슬랙 연동)
  - **Rationale:** 즉각적인 CS 효율성 개선 (90% 자동화 목표)

- **위기 고객 대응 SOP:**
  - 부정 리뷰 감지 시 1:1 메시지 발송 가이드
  - 불량 사유 태그 수동 입력
  - **Rationale:** 평판 보호 및 고객 충성도 전환

**공통 인프라**

- **데이터베이스:**
  - 리드 정보 저장
  - 리뷰/문의 데이터 저장 및 감성 분석 결과
  - **Rationale:** 데이터 기반 의사결정의 기초

- **알림 시스템:**
  - 슬랙 연동 (리드 확보, 위기 고객 알림)
  - **Rationale:** 실시간 대응 가능

### Out of Scope for MVP

- **고급 웹 개인화:** 행동 기반 실시간 콘텐츠 변경 (Phase 2)
- **AI 연관 추천 엔진:** Cross-selling 자동 추천 (Phase 2)
- **'살아있는' 상세 페이지:** 리뷰 기반 자동 F.A.Q 생성 및 A/B 테스트 (Phase 2)
- **글로벌 이커머스 확장:** Amazon, Shopify 채널 (Phase 2)
- **경쟁사 자동 광고비 조정:** 품절 감지 시 자동 광고비 증액 (Phase 3)
- **다국어 확장:** 베트남어, 태국어 등 (Phase 2)

### MVP Success Criteria

**MVP는 다음을 달성하면 성공:**

1. **B2B 리드 자동 수집:** 월 20개 이상의 검증된 B2B 리드 확보

3. **위기 대응 시간:** 부정 리뷰 감지 후 1시간 이내 1차 대응 달성률 90%
4. **모니터링 안정성:** 10분 간격 감지 성공률 95% 이상 (다운타임 최소화)
5. **사용자 피드백:** 챗봇 사용자 만족도 조사 4.0/5.0 이상

## Risks & Open Questions

### Key Risks

- **AI 답변 품질 저하:** LLM이 부적절하거나 부정확한 답변 생성 시 고객 불만 및 평판 손상
  - **완화 전략:** 답변 템플릿 사전 학습, 민감한 질문은 사람에게 에스컬레이션

- **웹 스크래핑 법적 문제:** 쿠팡, 스마트스토어가 스크래핑 금지 정책 시행 시 모니터링 중단 위험
  - **완화 전략:** robots.txt 준수, 공식 API 출시 시 즉시 전환, 법적 자문 확보

- **AI API 비용 폭증:** 트래픽 증가 시 OpenAI/Anthropic API 비용 예산 초과
  - **완화 전략:** 캐싱 전략, 응답 길이 제한, 사용량 모니터링 및 알림

- **경쟁사 모방:** 경쟁사가 유사한 AI 자동화 도입 시 차별화 상실
  - **완화 전략:** 3M 공식 파트너십 강조, 데이터 축적으로 AI 정확도 향상, 빠른 기능 확장

- **기술 의존도:** AI 모델 제공사의 정책 변경(API 가격 인상, 서비스 중단) 시 사업 연속성 위험
  - **완화 전략:** 복수 AI 제공사 연동 준비, 자체 모델 Fine-tuning 장기 계획

### Open Questions

- **챗봇 언어 톤:** 공식적(Formal) vs. 친근함(Casual) - B2B 파트너와 B2C 고객 각각 어떤 톤이 적절한가?
- **견적 범위 정확도:** 예상 견적 범위가 실제 견적과 20% 이상 차이 시 고객 신뢰도 하락 우려 - 어느 정도 범위가 적절한가?
- **위기 고객 보상 기준:** 소정의 보상(예: 이어폼 1쌍)의 구체적 기준은? 모든 부정 리뷰에 제공할 것인가?
- **경쟁사 모니터링 윤리:** 경쟁사 정보 수집의 윤리적 경계는? 어디까지 허용되는가?
- **글로벌 확장 우선순위:** Amazon(미국), Shopify(유럽), 또는 동남아 플랫폼(Lazada, Shopee) 중 어디를 먼저 공략할 것인가?

### Areas Needing Further Research

- **E-commerce API 가용성:** 쿠팡, 스마트스토어 공식 Partner API 신청 가능 여부 확인
- **AI Fine-tuning ROI:** 범용 LLM vs. 자체 Fine-tuned 모델의 비용 대비 성능 비교
- **경쟁사 벤치마크:** 유사 AI 자동화 도입 사례 조사 (글로벌 B2B 유통사)
- **법적 검토:** 웹 스크래핑의 한국 법률 적합성 (불공정거래행위 여부)
- **고객 수용도 테스트:** MVP 전 챗봇 프로토타입으로 실제 고객 반응 테스트 (A/B 테스트)
